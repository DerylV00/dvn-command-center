What Goes Where
After Each Type of Event
🤝
Ken Conversation
After Every Ken Meeting
  • Notion → Ken Wilkins relationship page → new conversation entry
  • Notion → DVN Tasks → new tasks for each action item extracted
  • data.json → projects.ken_alignment.notes → prepended
  • data.json → projects.ken_alignment.next_action → updated
  • data.json → projects.ken_alignment.last_conversation → today's date
  • Dashboard → Ken Alignment section reflects new status instantly
💪
Health Update
After Weigh-Ins or Lab Updates
  • Notion → DVN Health Log → new entry with date, weight, energy, sleep
  • data.json → projects.health.weight_current → updated number
  • data.json → projects.health.weight_logs → new entry appended
  • Dashboard → Physical Performance section → weight bar moves
  • Dashboard → sidebar stats → weight number updates
  • Optimus flags if no health update in 5+ days
💰
Financial Update
After Income or Debt Payments
  • Notion → DVN Financial Tracker → new entry with type, amount, category
  • data.json → projects.financial.debt_total → reduced if payment made
  • data.json → projects.financial.cash_on_hand → updated
  • Dashboard → Financial progress bars move visually
  • Optimus generates monthly summary on the 1st of each month
  • Celebrates visible progress — the bars moving matters psychologically
🔥
Voice Dump (General)
After Any Shift, Thought, or Event
  • Optimus classifies: WIN / INSIGHT / FRICTION / INTENTION
  • Optimus identifies which project it belongs to
  • Notion → DVN Daily Log → new entry with type, project, content
  • data.json → growth_log → new entry prepended (newest first)
  • Dashboard → Growth Log section shows new entry immediately
  • If shadow detected → Optimus names it and suggests the counter

Optimus Processes
Drop the JSON into Optimus. It extracts: key decisions, action items with dates, commitments from both sides, relationship temperature, business ideas. Creates tasks in Notion automatically.
→

Step 3
Claude Rebuilds
Drop the MD into Claude for deep debrief. We analyze what was said, update strategy files, rebuild any documents that need structural changes. Then Claude hands new files back to upload.

The rule: MD to Claude for analysis and strategy. JSON to Optimus for logging and task creation. Both happen after every recorded conversation with Ken.

The key difference between Notion and the Dashboard: Notion is the permanent archive — deep, searchable, historical. The Dashboard is the daily surface — simple, visual, current. Notion holds everything. The dashboard shows what matters today.
